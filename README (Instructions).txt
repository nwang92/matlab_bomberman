BOMBERMAN
E177 Spring 2013 Final Project
Created by: Nelson Wang, Tianyu Wang, Peter Kwon



PLAYER 1 CONTROLS:
W - Move up
A - Move left
S - Move down
D - Move right
E - Place bomb



PLAYER 2 CONTROLS:
UP ARROW - Move up
LEFT ARROW - Move left
DOWN ARROW - Move down
RIGHT ARROW - Move right
Enter/Return - Place bomb



GAME CONTROLS:
Q - Quit game
Close window - Quit game
